Outils de Pentest

Reconnaissance et Énumération Initiale

Nmap

Scanner de ports et découverte de services.

# Déjà installé sur Kali

AutoRecon

Automatise l'énumération initiale (ports, services, vulnérabilités de base).

git clone https://github.com/Tib3rius/AutoRecon.git

RustScan

Scanner de ports extrêmement rapide.

git clone https://github.com/RustScan/RustScan.git; cd RustScan; cargo build --release

dirsearch

Outil pour découvrir des répertoires et fichiers web.

git clone https://github.com/maurosoria/dirsearch.git

GoBuster

Alternative à dirsearch, également très efficace (écrit en Go).

go install github.com/OJ/gobuster@latest

ffuf

Fuzzer web puissant et rapide (écrit en Go).

go install github.com/ffuf/ffuf@latest

WhatWeb

Identifie les technologies utilisées par un site web.

# Déjà installé sur Kali

Nikto

Scanner de vulnérabilités web.

# Déjà installé sur Kali

dnsrecon

Outil d'énumération DNS.

# Déjà installé sur Kali

enum4linux-ng

Énumération d'informations à partir de systèmes Windows via SMB (successeur d'enum4linux).

git clone https://github.com/cddmp/enum4linux-ng.git

smbclient

Client pour accéder aux partages SMB.

# Déjà installé sur Kali

rpcclient

Utilitaires pour interagir avec les services RPC.

# Déjà installé sur Kali

ldapsearch

Utile pour interroger LDAP (Active Directory).

# Déjà installé sur Kali (paquet ldap-utils)
Privilege Escalation (Linux)

LinPEAS

Script d'énumération complet pour les vulnérabilités d'escalade de privilèges sous Linux.

wget https://github.com/peass-ng/PEASS-ng/releases/latest/download/linpeas.sh -O linpeas.sh; chmod +x linpeas.sh

pspy

Outil pour surveiller les processus sous Linux.

wget https://github.com/DominicBreuker/pspy/releases/latest/download/pspy64 -O pspy64; chmod +x pspy64 wget https://github.com/DominicBreuker/pspy/releases/latest/download/pspy32 -O pspy32; chmod +x pspy32

GTFOBins

Collection de binaires Unix qui peuvent être utilisés pour contourner les restrictions de sécurité.

# Site web : https://gtfobins.github.io/

searchsploit

Base de données d'exploits locaux et distants.

# Déjà installé sur Kali

linux-kernel-exploits

Répertoire d'exploits pour le kernel Linux.

git clone https://github.com/SecWiki/linux-kernel-exploits.git

kernel-exploits

Autre répertoire d'exploits pour le kernel Linux.

git clone https://github.com/lucyoa/kernel-exploits.git
Privilege Escalation (Windows)

WinPEAS

Outil pour l'énumération des privilèges sous Windows.

wget https://github.com/peass-ng/PEASS-ng/releases/latest/download/winPEASany.exe -OutFile winPEASany.exe

AccessChk

Utilité en ligne de commande pour afficher les autorisations.

# Télécharger le Sysinternals Suite : https://learn.microsoft.com/fr-fr/sysinternals/downloads/sysinternals-suite

PowerUp

Script PowerShell pour identifier les vecteurs d'escalade de privilèges Windows.

git clone https://github.com/HarmJ0y/PowerUp.git

Seatbelt

Autre outil PowerShell pour l'énumération de la sécurité locale de Windows.

git clone https://github.com/GhostPack/Seatbelt.git

WES-NG

Base de données hors ligne des exploits Windows basée sur les informations systeminfo.

git clone https://github.com/bitsadmin/wesng.git

SharpUp

Version C# de PowerUp (nécessite .NET sur la cible).

git clone https://github.com/GhostPack/SharpUp.git

LOLBAS

Collection de binaires, scripts et librairies Windows signés utilisables pour des actions malveillantes.

# Site web : https://lolbas-project.github.io/
Active Directory Exploitation

BloodHound

Outil pour visualiser les relations dans Active Directory et identifier les chemins d'attaque.

git clone https://github.com/BloodHoundAD/BloodHound.git git clone https://github.com/BloodHoundAD/SharpHound.git

Impacket

Collection d'outils Python pour travailler avec les protocoles réseau (inclut des outils AD).

git clone https://github.com/fortra/impacket.git

CrackMapExec (CME)

Boîte à outils post-exploitation pour tester et abuser des environnements Windows/Active Directory.

git clone https://github.com/byt3bl33d3r/CrackMapExec.git

Mimikatz

Outil puissant pour extraire les mots de passe, les tickets Kerberos, etc.

# Chercher l'exécutable sur GitHub (souvent dans des dépôts comme gentilkiwi/mimikatz)

Rubeus

Ensemble d'outils C# pour l'interaction brute avec Kerberos.

git clone https://github.com/GhostPack/Rubeus.git

PowerSploit

Framework PowerShell post-exploitation (inclut PowerView pour l'énumération AD).

git clone https://github.com/PowerShellMafia/PowerSploit.git

evil-winrm

Shell WinRM pour une utilisation offensive.

gem install evil-winrm

Kerbrute

Outil pour la force brute des noms d'utilisateurs et des mots de passe Kerberos.

git clone https://github.com/ropnop/kerbrute.git; cd kerbrute; go build

ADAPE-Script

Script PowerShell pour automatiser certaines tâches d'énumération et d'évaluation AD.

git clone https://github.com/hausec/ADAPE-Script.git
Outils de Pivoting et Transfert de Fichiers

Ligolo-ng

Outil de pivoting moderne pour le tunneling.

Installation Agent

wget https://github.com/nicocha30/ligolo-ng/releases/latest/download/ligolo-ng_agent_linux_amd64.tar.gz

Installation Proxy

wget https://github.com/nicocha30/ligolo-ng/releases/latest/download/ligolo-ng_proxy_linux_amd64.tar.gz

Chisel

Outil de tunneling TCP/UDP sur HTTP.

wget https://github.com/jpillora/chisel/releases/latest/download/chisel_linux_amd64.gz; gunzip chisel_linux_amd64.gz; chmod +x chisel_linux_amd64

Sshuttle

VPN transparent via SSH.

sudo apt-get install sshuttle

socat

Utilité de relais réseau polyvalente.

sudo apt-get install socat

Netcat (nc)

Utilité réseau pour lire et écrire des données à travers des connexions réseau.

# Déjà installé sur Kali (souvent `netcat-traditional` ou `ncat`)

ncat

Version moderne de Netcat avec plus de fonctionnalités.

sudo apt-get install ncat

Python SimpleHTTPServer / http.server

Serveur web simple pour le transfert de fichiers.

python -m SimpleHTTPServer 80 (Python 2) python3 -m http.server 80 (Python 3)

Impacket smbserver.py

Mettre en place un serveur SMB pour le transfert de fichiers.

# Nécessite Impacket : python3 -m pip install impacket sudo python3 /opt/impacket/examples/smbserver.py share /path/to/share
