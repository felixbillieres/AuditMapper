AuditMapper Security Assessment Suite

Suite d'outils professionnels pour audits de sécurité et tests d'intrusion. Gérez vos hosts, générez des rapports et optimisez vos pentests avec une interface moderne et intuitive.

🚀 Commencer maintenant 📖 Découvrir les outils

Fonctionnalités Principales

Des outils puissants conçus pour les professionnels de la cybersécurité

🖥️

Host Manager

Gérez vos hosts de manière centralisée avec une interface graphique interactive. Organisez par catégories, ajoutez des notes et générez des rapports automatiquement.

🌐

Config Generator

Générez automatiquement vos fichiers de configuration (hosts, Kerberos, ProxyChains) à partir de vos données d'audit.

🔍

Grep Master

Parser intelligent pour extraire les informations utiles de vos outputs (secretsdump, SAM, LSASS, etc.).

🔀

Pivot Master

Générateur de commandes de pivoting avec visualisation réseau interactive pour vos mouvements latéraux.

🔑

Guide Privesc

Checklist interactive pour l'escalade de privilèges avec suivi de progression et génération de rapports.

📊

Rapport Generator

Créez des rapports de vulnérabilités professionnels avec templates personnalisables et export multi-format.
